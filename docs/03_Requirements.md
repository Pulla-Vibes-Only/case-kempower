# LightHead3000Smart+ Requirements

| REQ-ID      | Requirement Description | Type  | Priority |
| :---        |    :----   |       :----   |        :---: | 
| **001** | The system works reliably as intented (other requirements are met) | Functional | Must |
| **002** | The guest is able to choose the operating system language | Functional | Must |
| **003** | The system is energy efficient (for example, lights stay off if room is empty) | Non-Functional | Must |
| **004** | The system control device (touch screen) clearly indicates the status of every light in the room | Functional | Should |
| **005** | Every room has its own controlling system and every room can be controlled through hotel central system by hotel personnel | Functional | Must |
| **006** | Room lights can be used to indicate an emergency | Functional | Should |
| **007** | The system must be easy to install and scalable | Functional | Should |
| **008** | The wall bracket of control device has a wireless charging | Functional | Must |
| **009** | The brightness, color and color temperature can be adjusted | Functional | Should |
| **010** | The control device can be removed from wall mount | Functional | Should | 
| **011** | The doorway has IR sensors to monitor if the room actually has people inside | Functional | Could |
| **012** | The control device displays possible rooms (bathroom, kitchen etc) separately | Functional | Should |
| **013** | The system has pre-installed lightning setups for different moods and situations | Functional | Could |
| **014** | The guest can control the lights through an app even before entering the room | Functional | Could |
| **015** | The control device has a map of the hotel room | Functional | Could |
| **016** | The staff (janitorial personnel for example) can define a preferred lightning setup (for cleaning for example) through their own app | Functional | Could |
| **017** | The lights adjust automatically based on time of day and the amount of natural light | Functional | Could |
| **018** | The hotel manager is able to monitor the electricity consumption rate of the whole light system | Non-Functional | Could |
